<!DOCTYPE html>

  </body>
</html>

La etiqueta <!DOCTYPE html> indica al navegador que se trata de un documento HTML5. La etiqueta <html> es la raíz del documento y contiene todos los elementos de la página web. La etiqueta <head> contiene información sobre la página, como el título de la página y los metadatos. La etiqueta <body> contiene el contenido visible de la página web, como títulos, párrafos y enlaces.

La meta etiqueta <meta name="viewport" content="width=device-width, initial-scale=1.0"> indica al navegador que el contenido de la página se debe adaptar al ancho del dispositivo en el que se visualiza. Esto ayuda a asegurar que la página se muestre correctamente en dispositivos móviles con diferentes resoluciones de pantalla.
