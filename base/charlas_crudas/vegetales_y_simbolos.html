El símbolo de exclamación se utiliza en matemáticas para denotar el factorial de un número. El factorial de un número es el producto de todos los números enteros positivos desde 1 hasta el número dado. Por ejemplo, el factorial de 5 se escribe como 5! y se calcula como 5 x 4 x 3 x 2 x 1 = 120. El símbolo de exclamación se utiliza también en lógica matemática para denotar el operador lógico negación. Por ejemplo, "A!" se refiere a la negación de la afirmación A.
